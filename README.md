# gdx-tiledmap

This is a **fork of the [libGDX](https://github.com/libgdx/libgdx) library**, focusing exclusively on the **Tiled map** handling portion (`gdx-maps` / tiled map loaders, renderers, and related utilities).

The original code has been extracted from the libGDX monorepo to allow independent development, cleanup, and experimentation. Future work may include:

- Code cleanup and modernization
- API improvements and simplifications
- Bug fixes and maintenance
- Potential partial or full rewrites of specific components

All original credit goes to the [libGDX authors and contributors](https://github.com/libgdx/libgdx). This project remains licensed under the Apache License 2.0.
